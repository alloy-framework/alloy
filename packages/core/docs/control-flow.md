# Control Flow

- [**`<Show>`**](api/components/Show.md) — conditional rendering. Renders `children` when `when` is truthy, otherwise `fallback`.
- [**`<For>`**](api/components/For.md) — iteration with joiner props (`comma`, `hardline`, etc.).
- [**`<Switch>`**](api/components/Switch.md) / [**`<Match>`**](api/variables/Match.md) — multi-branch conditional. Use `<Match else>` for default.
